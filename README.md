Sistema de Formulário Web
Sistema de Formulário Responsivo com HTML e CSS

Este repositório foi criado com o objetivo de praticar a construção de formulários web utilizando HTML5 e CSS3. O projeto foca na estruturação correta de campos, organização semântica e validação básica de dados.

Objetivo do Projeto

O projeto tem como finalidade desenvolver habilidades em:

Estruturação semântica com HTML

Estilização moderna com CSS

A proposta foi simular um formulário real que poderia ser integrado futuramente a um back-end.
